# CareerCollection — 求職データベーステンプレート

> 日本企業への求職活動を Claude Code で管理する個人データベースです。
> 使用 Claude Code 管理日本企业求职活动的个人数据库模板。

---

## はじめに / 开始使用

### Step 1：个人信息设置

编辑 `profile.md`，填写你的学历・职历・语言・目标岗位信息。
Claude 会在每次任务开始前自动读取这个文件。

### Step 2：添加第一家公司

在 Claude Code 中输入：
```
○○という会社を調べて追加してください
```
Claude 会自动：
- 调查公司基本信息・招募情况・企业文化
- 根据你的 profile 评估有效路线并分配 Tier
- 创建 `companies/[企業名].md`
- 更新 `README.md` 和 `tracker.md`

### Step 3：更新求职进度

```
○○に応募しました
○○の面接が通過しました
```
状态变为「応募済」后，Claude 会自动收集面试经验并制作面试方案。

---

## Tier 分类说明

| Tier | 意味 |
|------|------|
| A | 有效路线 2 条以上，现阶段最优先 |
| B | 有 1〜2 条有效路线，存在障壁但可突破 |
| C | 求人依赖型，有空缺时立刻行动 |
| D | 中长期目标，需积累经验后再挑战 |

## 主要命令一览

| 操作 | 说明 |
|------|------|
| 「○○を調べて追加して」 | 调查并新增企业 |
| 「○○に応募しました」 | 更新应募状态 → 触发面试准备 |
| 「情報を更新して」 | 更新所有企业的最新动态 |
| 「Excelに出力して」 | 生成 tracker Excel 文件 |
